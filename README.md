# GNOME Pomodoro DMS Theme Sync

Sync GNOME Pomodoro's `dark-theme` plugin with Dank Material Shell (DMS) theme mode.

> **DMS-only:** This implementation depends on `dms ipc theme getMode`.  
> If you use another desktop environment/shell, you will need your own theme-source integration.

Requires: `dms`, `gnome-pomodoro`, `python3`, `gsettings`, and `systemd --user`.

## Install / Uninstall

```bash
./install.sh
./uninstall.sh
```
